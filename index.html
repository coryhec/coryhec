HiveConnect

open source

A commissioned Go project built on gophertunnel for private competitive matchmaking on Hive. Originally client work that became open source when the client no longer needed it. My main goal was using a real brief to learn Go properly — the core is complete with a packet response system, in-game command execution, a packet logger, and a REST API to track bot state (on my personal build, wired up to a Discord bot). Unfinished, but the architecture is solid. A friend, aShanki, has since built a new version on the same core.

about

I'm a developer who got serious about building things early. At 14 I founded Plasmic Client, a Minecraft Bedrock mod that reached 10,000 members and ~100k YouTube views before I handed it off — it became what is now Onix Client. At 15 I launched Impulse Ranked, solo-coding the entire bot and running the community for 3 years.

Those projects taught me things no tutorial covers: how to keep a community's trust, how to ship under pressure, and how fast things fall apart when you go quiet on your users. That shapes how I build everything now. Currently open to new work — reach out on GitHub.
